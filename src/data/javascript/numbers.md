## Different number methods in javascript :

```javascript
var x = 13
```

1. return a number into a string

```javascript
var str = x.toString()
```

1. to Fixed

```javascript
var x = 0.95477
var y = x.toFixed(2)
```

1. convert a number to an integer

```javascript
var x = 0.9577
var y = x.parseInt()
```
